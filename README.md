# ☕ Coffee Dev Experience

Uma landing page moderna de cafeteria inspirada no universo dos desenvolvedores, unindo **arte do café + produtividade + tecnologia**.

---

## 🚀 Sobre o Projeto

O **Coffee Dev Experience** foi desenvolvido com o objetivo de criar uma experiência visual premium, simulando um produto real de startup.

A proposta é representar um ambiente onde desenvolvedores podem trabalhar, estudar e se inspirar, com foco em:

- ☕ Experiência do café
- 💻 Produtividade
- 🎨 Design moderno
- 📱 Responsividade

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3 (externo)  
- JavaScript  
- Flexbox & Grid  
- Media Queries (Mobile First)  

---

## 📱 Funcionalidades

- ✅ Layout responsivo (Mobile First)
- ✅ Animações ao rolar (scroll reveal)
- ✅ Dark Mode automático e manual
- ✅ Navbar dinâmica (efeito ao rolar)
- ✅ Cards interativos (hover)
- ✅ Tabela de produtos
- ✅ Navegação entre páginas
- ✅ Link externo (redes sociais)
